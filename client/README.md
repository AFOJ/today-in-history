## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/AFOJ/today-in-history.git
```

2. Change to the client directory:

```bash
cd today-in-history/client
```

3. Install dependencies:

```bash
npm install
```

4. Copy the .env.example file and update the secrets.

```bash
cp .env.example .env
```

5. Run the project

```bash
npm run dev
```
